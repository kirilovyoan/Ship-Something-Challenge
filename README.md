# 56K — The Dial-Up Internet Revival App

**Team:** Yoan, Angel and Daniela

## What this is
A retro-styled landing page for "56K" — a fictional app that revives the dial-up internet experience. The site features CRT scanline effects, a CSS-animated modem connection simulation, glitch text effects, and a dark cyberpunk aesthetic that evokes 90s internet nostalgia. Built entirely with HTML and CSS — no JavaScript required.

## Live preview
Open `index.html` in a browser (or use VS Code Live Server).
